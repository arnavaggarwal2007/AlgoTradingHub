## Step 1 — Buy Hostinger Plan + Domain

1. Go to **hostinger.com** → choose **Business Web Hosting** (includes free domain, SSL, CDN)
2. During checkout, register your domain (e.g., `algotradingHub.com`)
3. Keep your Hostinger control panel login safe

> **Why Business plan?** Unlimited bandwidth, free domain, free SSL, LiteSpeed cache, and object cache (Redis-lite) — all crucial for a fast, high-ranking site.

---

## Step 2 — Install WordPress (One-Click)

1. Log in to **hPanel** (Hostinger control panel)
2. Go to **Websites** → **Add Website** → **WordPress**
3. Fill in:
   - **Site title**: AlgoTrading Hub (or your brand)
   - **Admin email**: your@email.com
   - **Admin username**: choose a strong username (NOT "admin")
   - **Password**: 20+ character password
4. Click **Install**. Done in ~30 seconds.
5. WordPress admin is at: `https://yourdomain.com/wp-admin`

## Step 4 — Choose a Theme (SEO + Speed Optimized)

### Recommended: **Astra** (free, fastest WordPress theme)

1. **Appearance → Themes → Add New** → Search "Astra" → Install → Activate
2. **Appearance → Customize** → configure colors/fonts to your brand

**Why Astra?**
- Loads in < 0.5s (critical for Google Core Web Vitals)
- Starter templates available (trading/finance templates)
- Works perfectly with Yoast SEO

### Alternative themes:
- **GeneratePress** — ultralight, developer friendly
- **OceanWP** — more visual options, slightly heavier

```
Primary Color: #1a73e8  (professional blue)
Heading Font: Inter or Roboto

```

## Step 5 — Configure Yoast SEO

1. Go to **SEO (Yoast) → General → Configuration Wizard**
2. Select:
   - "This is a company" → enter your site name
   - Upload logo (512x512 PNG)
3. **SEO → General → Features**:
   - ✅ XML sitemaps
   - ✅ Schema markup
   - ✅ Social previews
4. **SEO → Social** → add your Twitter/LinkedIn URLs
5. Submit sitemap to Google Search Console:
   - URL: `https://yourdomain.com/sitemap_index.xml`
   - Add to [Google Search Console](https://search.google.com/search-console)

## Step 8 — Link WordPress to Your Digital Ocean App

### Option A: Prominent Button on Home Page (Easiest)
```html
<!-- Add to Home page with HTML block or Astra header button -->
<a href="https://app.yourdomain.com" 
   target="_blank" 
   rel="noopener"
   class="wp-block-button__link">
  Launch Live Dashboard →
</a>
```

### Option B: Embed Dashboard in WordPress Page (iframe)
```html
<!-- WordPress page with Custom HTML block -->
<style>
  .dashboard-frame {
    width: 100%;
    height: 85vh;
    border: none;
    border-radius: 8px;
    box-shadow: 0 4px 20px rgba(0,0,0,0.15);
  }
</style>
<iframe 
  src="https://app.yourdomain.com" 
  class="dashboard-frame"
  title="Pre-Swing Trade Analysis Dashboard"
  loading="lazy">
</iframe>
```
> Note: For iframe embedding to work, your Digital Ocean Dash app must NOT send `X-Frame-Options: DENY`. Set it to `SAMEORIGIN` or remove it.

### Option C: Subdomain Navigation (Professional)
Configure your DNS on Hostinger:
```
yourdomain.com       → Hostinger WordPress
app.yourdomain.com   → Digital Ocean droplet IP  
www.yourdomain.com   → CNAME to yourdomain.com
```

**DNS Steps in Hostinger hPanel:**
1. **Domains → DNS Zone Editor**
2. Add A record:
   - Host: `app`
   - Points to: `YOUR_DO_DROPLET_IP`
   - TTL: 300

## Step 9 — WordPress Application Password (for Auto-Blog API)

This allows the auto_blog scheduler to publish posts:

1. WordPress Admin → **Users → Profile** → scroll to bottom
2. Find **Application Passwords** section
3. Enter name: `AutoBlog Bot`
4. Click **Add New Application Password**
5. **Copy the password now** (shown only once!)
6. Add to your `.env` file:
```env
WP_SITE_URL=https://yourdomain.com
WP_USERNAME=your_wp_admin_username
WP_APP_PASSWORD=xxxx xxxx xxxx xxxx xxxx xxxx
```

## Step 10 — SEO-Optimized About/Home Content Structure

For a trading education site, Google ranks you based on **E-E-A-T**:
- **Experience**: Show you've actually traded
- **Expertise**: Demonstrate technical knowledge
- **Authoritativeness**: Backlinks, guest posts
- **Trustworthiness**: SSL, disclaimers, accurate info

**Key SEO Actions:**
```
1. Install Yoast → fill all "Snippet Preview" fields for every page
2. Homepage title: "Free Swing Trade Signals | [Brand] — AI-Powered Analysis"
3. Submit sitemap to Google Search Console
4. Add Google Analytics 4 via MonsterInsights
5. Enable schema markup in Yoast (Article schema for blog posts)
6. Target long-tail keywords: "EMA swing trading strategy Python", "best stocks to swing trade"
7. Internal link every blog post to 2-3 other posts + home page
```

## Step 11 — Speed Optimization (Core Web Vitals)

Google uses page speed as a direct ranking factor.

**LiteSpeed Cache Setup:**
1. Plugin → LiteSpeed Cache → Settings
2. ✅ Enable Cache
3. ✅ Browser Cache
4. ✅ Image Optimization → enable WebP
5. ✅ CSS/JS Minification + Combination

**Expected result:** PageSpeed score of 90+ (mobile), 95+ (desktop)
